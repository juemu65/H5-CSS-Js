##  段落与换行
+ 段落的前后必须是空行  
  
+ 如果需要在段落内加入换行 `<br>`

    示例:
    
    ```
    Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。<br>它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或者 HTML）文档”。
    ```
    Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。<br>它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的 XHTML（或者 HTML）文档”。
